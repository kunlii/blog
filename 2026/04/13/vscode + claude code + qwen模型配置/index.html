相较于浏览器端对话，VS Code中的Claude Code插件能显著提升开发效率。为规避Claude API的成本与风险，可通过以下步骤将其配置为调用阿里云Qwen API，实现稳定且经济的编程辅助。

  1. 安装Claude Code插件：打开VS Code，进入扩展市场（快捷键 Ctrl+Shift+X），搜索并安装 Anthropic 官方发布的 Claude Code 插件。
  2. 打开用户配置文件：按下 Ctrl+Shift+P 打开命令面板，输入 Open User Settings (JSON) 并选择，确保打开的是用户级配置文件（非工作区），避免配置仅对单个项目生效。
  3. 编辑配置文件：在 settings.json 文件中添加以下配置（若文件已有内容，需在末尾配置后添加英文逗号 , 再粘贴）：
  1. 替换说明：将 YOUR_API_KEY 替换为从 阿里云百炼平台 申请的 Qwen API密钥。确保 selectedModel 与 ANTHROPIC_MODEL 值一致（如使用其他模型，同步修改为 qwen-coder-plus 等）。在写配置文件的时候我注意到vscode有提示说第一行的claudeCode.selectedModel是未定义配置项，猜测可能没这个也行，
  2. 保存并重启VS Code： 保存配置文件后，重启编辑器使配置生效。
